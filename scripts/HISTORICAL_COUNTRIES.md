# 历史消亡国家代码处理规则

## 背景

TMDB 对已消亡国家使用非现行 ISO 3166-1 代码，地图无法直接点亮。
本项目按「文化归属」口径将其映射到现代继承国。

## 硬编码默认映射（可套用，扩展时直接沿用）

| TMDB 代码 | 历史国家 | 默认映射 | 依据 |
|-----------|---------|---------|------|
| `XC` | 捷克斯洛伐克 | `CZ` 捷克 | TMDB 惯例；绝大多数 XC 片为捷克语/斯洛伐克语，映射捷克为通行做法 |
| `SU` | 苏联 | `RU` 俄罗斯 | 苏联电影工业主体在俄罗斯，继承国映射合理 |
| `DD` | 东德 | `DE` 德国 | 两德统一，文化归属无争议 |
| `CS` | 塞尔维亚和黑山 | `RS` 塞尔维亚 | 黑山独立后主体在塞尔维亚 |

## ⚠️ 不可硬编码映射（必须逐部人工核实）

### `YU` — 南斯拉夫

南斯拉夫解体为多个继承国，不同影片应按**语言、导演民族属、出品公司所在地**逐部确认：

| 继承国 | 代码 | 语言 / 民族 |
|--------|------|------------|
| 塞尔维亚 | `RS` | 塞尔维亚语、塞族导演 |
| 克罗地亚 | `HR` | 克罗地亚语、克族导演 |
| 斯洛文尼亚 | `SI` | 斯洛文尼亚语 |
| 北马其顿 | `MK` | 马其顿语 |
| 波黑 | `BA` | 波斯尼亚语 |
| 黑山 | `ME` | 黑山语 |

**在 `enrich_countries.py` 的 `MANUAL_OVERRIDE` 字典里逐条覆盖，不可套用默认。**

## 扩展到其他奖项时的操作流程

1. 运行 `scan_historical.py` 扫描新增数据中的非现存国家代码
2. `XC`/`SU`/`DD`/`CS` → 直接套默认映射，无需处理
3. `YU` → 查导演姓名 + 语言，在 `MANUAL_OVERRIDE` 逐条添加
4. 新出现其他代码 → 先查 Wikipedia 确认继承国，再决定是否加入硬编码或逐部处理
